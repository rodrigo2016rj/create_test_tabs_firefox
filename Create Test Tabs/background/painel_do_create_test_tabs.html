Create Test Tabs

Instructions:

1) Just below these instructions, register a page of your system that has a form.

2) In the item of the page you registered, click the "Expand options" button.

3) Click the "Create new form input combinations" button to create a form input combination for the registered page.

4) One at a time, fill in the names and choose values ​​for your form entries. For each name and value pair, click the "Add" button to add. Entries not added will have an empty value when tabs are created by this addon.

5) Repeat steps 3 and 4 for all fill combinations you want to test for the registered page form.

6) Click the "Create Test Tabs" button and your browser will open a tab for each combination you create. Each tab will have the fills you set in each combination.

7) Now you just need to go to each tab and submit the form to see if the result found will be the same result you expect from your system.

Important: This addon is set to fill forms only from URLs that start with: "http://localhost/", "https://localhost/", "http://127.0.0.1/", "https://127.0.0.1/", "http://[::1]/" and "https://[::1]/".
